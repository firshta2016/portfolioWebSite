Pig Dice Game

This project is created in plain javascript. Instructions on how to play:

  • The game has 2 players, playing in rounds
  • In each turn, a player rolls a dice as many rimes as he wishes. Each result get added to his ROUND score
  • BUT, if the player rolls a number 1, all of the current score gets lost. After that, its the next players turn.
  • The player can choose to 'Hold', which means that the current ROUND score gets added to the GLOBAL score. After that, its the next player's turn.
  • The first player to reach 100 points on GLOBAL score wins the game.
